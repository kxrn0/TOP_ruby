# Chess

## Overview

When the program starts, a welcome message is printed, and then a menu for the options.
```
Chess.
Copyright kxrn0 1987

Menu:

(a) - Start a new game
(b) - Load saved game
(c) - Exit 

Enter choice > 
```
Choosing to start a new game will promt the users for their names, and set up a new game.

```
Chess.
Copyright kxrn0 1987

Menu:

(a) - Start a new game
(b) - Load saved game
(c) - Exit 

Enter choice > a

Enter the first player's name > Alice
Enter the second player's name > Bob

Alice will be white ♔
Bob will be black ♚

Game Menu

(a) - Start game
(b) - Save game
```
